    How to use

    📝 Tasks

    • Add — Type a task and press Add
    • Complete — Check the checkbox
    • Edit — Click ✏ to edit inline, press Enter to save
    • Delete — Click ✕
    • Reorder — Drag and drop tasks

    🏷️ Tags

    Use #tagname in any task. Each tag gets its own color automatically. Click a tag pill to filter tasks by that tag.

    📅 Due Dates

    Add a due date with @Due[today], @Due[tomorrow], or @Due[dd/mm/yy]. Overdue tasks are highlighted in red.

    🎨 Themes

    Click 🌙/☀️ to toggle between dark and light mode.

    ⏱️ Pomodoro Timer

    Use the timer for focus sessions (25 min focus / 5 min break). Click Pin to open a small always-on-top window that stays visible even when you switch tabs.
