    How to use

    Tasks

    • Add — Type in quick-add and press Enter, or click "Details" for the full form.
    • Complete — Check the checkbox.
    • Edit — Click ✏ to edit all fields.
    • Delete — Click ✕ (undo available via toast).
    • Reorder — Drag tasks when sorted by Custom.

    Tags & Filter

    Add tags in the task form. Click a tag pill to filter. Projects appear as project:NAME in the tag cloud.

    Due Dates & Repeat

    Set a due date and optional repeat frequency. Recurring tasks auto-recreate on completion.

    Golden Task

    Click ⭐ to mark a task as golden — it pins to the top with a special glow.

    Themes

    Toggle dark/light mode with the theme button.

    Pomodoro Timer

    25 min focus / 5 min break cycles. Click the Pin button for Picture-in-Picture mode.

    Keyboard Shortcuts

    n — Focus quick-add · Esc — Close modals
